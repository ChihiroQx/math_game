# 🇨🇳 Gitee Pages 部署指南（国内推荐）

## 为什么选择 Gitee Pages？

- ✅ **国内访问飞快** - 服务器在国内，秒开
- ✅ **完全免费** - 无任何费用
- ✅ **操作简单** - 类似 GitHub，中文界面
- ✅ **稳定可靠** - 国内最大代码托管平台

---

## 📦 第一步：构建游戏

双击运行 `deploy.bat`，生成 `dist` 文件夹

---

## 🚀 第二步：部署到 Gitee

### 1. 注册 Gitee 账号

访问：https://gitee.com/signup

- 填写用户名、邮箱、密码
- 验证邮箱
- 完成注册

### 2. 创建仓库

1. 登录后，点击右上角 **"+"** → **"新建仓库"**
2. 填写信息：
   - **仓库名称**：`math-game`（或任意英文名）
   - **路径**：自动生成（或自定义）
   - **仓库介绍**：数学童话冒险游戏
   - **是否开源**：选择 **"公开"**
   - 其他选项保持默认
3. 点击 **"创建"**

### 3. 上传游戏文件

**方法A：网页上传（推荐新手）**

1. 进入刚创建的仓库
2. 点击 **"上传文件"**
3. 打开 `dist` 文件夹
4. **全选所有文件和文件夹**（Ctrl+A）
5. 拖拽到网页上传区域
6. 等待上传完成（可能需要1-2分钟）
7. 填写提交信息：`初始化游戏`
8. 点击 **"提交"**

**方法B：使用 Git 命令**

```bash
cd dist
git init
git add .
git commit -m "部署数学游戏"
git remote add origin https://gitee.com/你的用户名/math-game.git
git push -u origin master
```

### 4. 启动 Gitee Pages

1. 在仓库页面，点击 **"服务"** → **"Gitee Pages"**
2. 选择分支：**master**
3. 选择目录：**/** (根目录)
4. 点击 **"启动"**
5. 等待几秒钟，部署完成！

### 5. 获取访问链接

部署成功后，会显示您的游戏地址：

```
https://你的用户名.gitee.io/math-game/
```

**立即复制链接，在浏览器打开试试！** 🎮

## 🔄 第四步：更新游戏

当您修改游戏后：

1. 重新运行 `deploy.bat`
2. 进入 Gitee 仓库
3. 点击 **"上传文件"**
4. 选择 **"覆盖同名文件"**
5. 上传新的文件
6. 进入 **Gitee Pages** 设置
7. 点击 **"更新"** 按钮
8. 等待几秒，更新完成！

## 🎨 绑定自定义域名（可选）

如果您有自己的域名：

1. 在 Gitee Pages 设置中
2. 点击 **"自定义域名"**
3. 输入您的域名（如 `game.yourdomain.com`）
4. 在域名服务商处添加 CNAME 记录
5. 等待生效（几分钟到几小时）

## ⚠️ 注意事项

### 1. 仓库必须公开
- Gitee Pages 只支持公开仓库
- 确保创建时选择"公开"

### 2. 首次部署需要审核
- 首次启动 Gitee Pages 可能需要实名认证
- 按提示完成认证即可（免费）

### 3. 更新后刷新
- 更新游戏后，需要在 Pages 设置点"更新"
- 用户访问时可能需要强制刷新（Ctrl+F5）清除缓存

### 4. 文件大小限制
- 单个仓库最大 1GB
- 单个文件最大 100MB
- 我们的游戏远小于这个限制

## ❓ 常见问题

**Q: 为什么页面打不开？**
A: 确保：
- 仓库是公开的
- Gitee Pages 已启动
- 等待1-2分钟让服务生效

**Q: 更新后没变化？**
A: 
1. 确保点击了 Pages 设置中的"更新"按钮
2. 清除浏览器缓存（Ctrl+Shift+Delete）
3. 强制刷新（Ctrl+F5）

**Q: 访问速度慢？**
A: Gitee Pages 在国内速度很快，如果慢可能是：
- 网络问题，换个网络试试
- 浏览器缓存问题，清除缓存

**Q: 需要付费吗？**
A: 完全免费！无任何隐藏费用

**Q: 会被删除吗？**
A: 不会，只要：
- 仓库不违规（我们的游戏完全合规）
- 长期不使用也不会删除
