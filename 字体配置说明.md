# 字体配置说明

## 概述

游戏现在支持灵活的字体配置系统，可以轻松更换字体。

## 字体配置位置

字体配置在 `src/config/FontConfig.ts` 文件中。

## 当前字体方案

### 默认字体（系统字体）
- **标题字体**：`Arial Black, Microsoft YaHei, "Microsoft YaHei UI", sans-serif`
- **正文字体**：`Microsoft YaHei, "Microsoft YaHei UI", Arial, sans-serif`
- **数字字体**：`Arial, "Courier New", monospace`

## 如何更换字体

### 方法一：使用系统字体（无需加载）

直接修改 `FontConfig.ts` 中的 `CURRENT_FONTS`：

```typescript
export const CURRENT_FONTS: FontConfig = {
  title: '你想要的字体, 备用字体, sans-serif',
  body: '你想要的字体, 备用字体, sans-serif',
  number: '你想要的字体, monospace'
};
```

### 方法二：使用 Google Fonts（需要网络）

1. **在 `index.html` 中取消注释字体加载代码**：

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;700;900&display=swap" rel="stylesheet">
```

2. **修改 `FontConfig.ts` 使用 Google Fonts**：

```typescript
export const CURRENT_FONTS: FontConfig = GOOGLE_FONTS;
```

### 方法三：使用自定义字体文件

1. **将字体文件放入 `assets/fonts/` 目录**

2. **在 `index.html` 中添加字体定义**：

```css
@font-face {
  font-family: 'YourFontName';
  src: url('assets/fonts/YourFont.woff2') format('woff2'),
       url('assets/fonts/YourFont.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}
```

3. **在 `FontConfig.ts` 中使用**：

```typescript
export const CURRENT_FONTS: FontConfig = {
  title: '"YourFontName", "Microsoft YaHei", sans-serif',
  body: '"YourFontName", "Microsoft YaHei", sans-serif',
  number: '"YourFontName", monospace'
};
```

## 推荐的字体选择

### 中文游戏推荐字体

1. **思源黑体（Noto Sans SC）**
   - Google Fonts 提供
   - 免费开源
   - 支持多种字重

2. **站酷字体**
   - 站酷快乐体（ZCOOL KuaiLe）- 适合儿童游戏
   - 站酷高端黑（ZCOOL KuaiLe）- 现代感

3. **系统字体**
   - Microsoft YaHei（微软雅黑）- Windows
   - PingFang SC（苹方）- macOS/iOS
   - Hiragino Sans GB（冬青黑体）- macOS

### 英文/数字推荐字体

1. **Arial Black** - 粗体标题
2. **Roboto** - Google 设计
3. **Open Sans** - 清晰易读

## 字体使用位置

字体配置会自动应用到：
- ✅ 所有按钮文字（通过 `ButtonFactory`）
- ⚠️ 场景中的文字需要手动更新（见下方）

## 更新场景中的字体

如果要在场景中使用新字体，需要导入并使用：

```typescript
import { getTitleFont, getBodyFont, getNumberFont } from '../config/FontConfig';

// 使用标题字体
this.add.text(x, y, '标题', {
  fontFamily: getTitleFont(),
  // ...
});

// 使用正文字体
this.add.text(x, y, '正文', {
  fontFamily: getBodyFont(),
  // ...
});
```

## 注意事项

1. **字体加载**：如果使用网络字体，确保在字体加载完成后再显示文字
2. **字体大小**：更换字体后可能需要调整字体大小
3. **性能**：自定义字体文件会增加加载时间
4. **兼容性**：确保字体在所有目标平台上可用

## 快速切换示例

### 切换到 Google Fonts（思源黑体）

1. 取消注释 `index.html` 中的 Google Fonts 链接
2. 在 `FontConfig.ts` 中：
```typescript
export const CURRENT_FONTS: FontConfig = GOOGLE_FONTS;
```

### 切换到站酷字体

1. 在 `index.html` 中添加：
```html
<link href="https://fonts.googleapis.com/css2?family=ZCOOL+KuaiLe&display=swap" rel="stylesheet">
```
2. 在 `FontConfig.ts` 中：
```typescript
export const CURRENT_FONTS: FontConfig = ZCOOL_FONTS;
```

## 测试字体

更换字体后，建议：
1. 清除浏览器缓存
2. 在不同设备上测试
3. 检查字体加载时间
4. 验证所有文字显示正常
