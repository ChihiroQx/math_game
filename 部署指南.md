# 🚀 游戏部署指南 - 让小朋友们在线玩

## 📦 第一步：构建游戏

### 1. 运行构建脚本

双击运行 `build.bat`，或者在命令行执行：

```bash
npm run build
```

构建完成后，会在 `dist` 文件夹中生成以下文件：
- `index.html` - 游戏主页面
- `bundle.js` - 游戏代码（已压缩优化）
- `assets/` - 所有游戏资源（图片、音频等）

---

## 🌐 第二步：选择部署平台

### 方案1：GitHub Pages（推荐 - 完全免费）

**优点**：
- ✅ 完全免费
- ✅ 无需域名（提供 `username.github.io/项目名` 访问）
- ✅ 支持自定义域名
- ✅ 稳定可靠

**步骤**：

1. **创建 GitHub 账号**（如果没有）
   - 访问 https://github.com
   - 点击 "Sign up" 注册

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名：`math-game`（或任意名称）
   - 设置为 Public（公开）
   - 点击 "Create repository"

3. **上传游戏文件**
   
   方法A：使用 GitHub 网页界面（简单）
   ```
   - 点击 "uploading an existing file"
   - 将 dist 文件夹中的所有文件拖拽上传
   - 点击 "Commit changes"
   ```

   方法B：使用 Git 命令（推荐）
   ```bash
   cd dist
   git init
   git add .
   git commit -m "部署数学游戏"
   git branch -M main
   git remote add origin https://github.com/你的用户名/math-game.git
   git push -u origin main
   ```

4. **启用 GitHub Pages**
   - 进入仓库的 "Settings"
   - 左侧菜单点击 "Pages"
   - Source 选择 "main" 分支
   - 点击 "Save"
   - 等待几分钟，访问：`https://你的用户名.github.io/math-game/`

---

### 方案2：Vercel（推荐 - 免费且超快）

**优点**：
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 全球 CDN 加速
- ✅ 部署速度极快

**步骤**：

1. **注册 Vercel 账号**
   - 访问 https://vercel.com
   - 使用 GitHub 账号登录

2. **部署项目**
   - 点击 "New Project"
   - 选择 "Import Git Repository"
   - 连接 GitHub 仓库
   - Framework Preset: 选择 "Other"
   - Output Directory: 输入 `dist`
   - 点击 "Deploy"

3. **获取访问链接**
   - 部署完成后，自动生成类似：`math-game.vercel.app` 的链接
   - 可以在设置中添加自定义域名

---

### 方案3：Netlify（免费 + 简单）

**优点**：
- ✅ 完全免费
- ✅ 拖放部署（最简单）
- ✅ 自动 HTTPS
- ✅ 免费域名

**步骤**：

1. **注册账号**
   - 访问 https://www.netlify.com
   - 点击 "Sign up"

2. **拖放部署**
   - 登录后，将整个 `dist` 文件夹拖到页面上
   - 自动部署，获得类似 `random-name-123456.netlify.app` 的链接

3. **自定义域名**（可选）
   - 在 Site settings → Domain management 中修改

---

### 方案4：国内平台（更快访问）

#### 4.1 Gitee Pages（国内访问快）

**步骤**：
1. 注册 https://gitee.com
2. 创建仓库并上传 dist 文件夹内容
3. 进入仓库 → 服务 → Gitee Pages
4. 选择分支，点击启动

#### 4.2 腾讯云静态网站托管

**步骤**：
1. 注册腾讯云账号
2. 进入"云开发 CloudBase"
3. 创建环境 → 静态网站托管
4. 上传 dist 文件夹

## 📱 让游戏支持手机（PWA）

创建 `dist/manifest.json`：

```json
{
  "name": "数学童话冒险",
  "short_name": "数学游戏",
  "description": "专为小朋友设计的数学学习游戏",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#87CEEB",
  "theme_color": "#FF69B4",
  "icons": [
    {
      "src": "icon-192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "icon-512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}
```

在 `dist/index.html` 的 `<head>` 中添加：

```html
<link rel="manifest" href="/manifest.json">
<meta name="theme-color" content="#FF69B4">
<meta name="apple-mobile-web-app-capable" content="yes">
```

这样小朋友就可以"添加到主屏幕"，像原生应用一样使用！

## ⚠️ 注意事项

1. **浏览器兼容性**
   - 建议使用 Chrome、Edge、Safari 等现代浏览器
   - 手机端支持 iOS Safari 和 Android Chrome

2. **加载速度**
   - 首次加载可能需要几秒（资源下载）
   - 后续访问会更快（浏览器缓存）

3. **数据存储**
   - 游戏进度保存在浏览器本地
   - 清除浏览器数据会丢失进度
   - 不同设备的进度不同步

4. **版权说明**
   - 确保游戏中使用的资源（图片、音效）有合法使用权
   - 建议在游戏中添加版权信息

## 🎉 快速部署命令（推荐）

创建一个自动化部署脚本 `deploy.bat`：

```batch
@echo off
echo 正在构建游戏...
call npm run build

echo.
echo 构建完成！
echo.
echo 请选择部署方式：
echo 1. GitHub Pages
echo 2. Vercel
echo 3. Netlify（拖放 dist 文件夹）
echo.
echo dist 文件夹已准备好，可以上传了！
pause
```
