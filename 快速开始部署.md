# 🚀 3分钟快速部署（最简单方法）

## 方法1：Netlify 拖放部署（推荐新手）

### 步骤：

1. **构建游戏**
   ```
   双击 deploy.bat
   ```

2. **访问 Netlify**
   - 打开 https://app.netlify.com/drop
   - 无需注册！

3. **拖放部署**
   - 将 `dist` 文件夹直接拖到网页上
   - 等待上传完成（约30秒）
   - 自动获得游戏链接！

4. **分享**
   - 复制链接发给小朋友
   - 或生成二维码分享

---

## 方法2：GitHub Pages（推荐长期使用）

### 步骤：

1. **构建游戏**
   ```
   双击 deploy.bat
   ```

2. **创建 GitHub 仓库**
   - 访问 https://github.com/new
   - 仓库名：`math-game`
   - 设为 Public
   - 创建

3. **上传文件**（网页操作，无需命令行）
   - 进入仓库
   - 点击 "uploading an existing file"
   - 打开 `dist` 文件夹
   - 全选所有文件（Ctrl+A）
   - 拖拽到网页上传
   - 点击 "Commit changes"

4. **开启 Pages**
   - 仓库 → Settings → Pages
   - Branch: 选择 `main`
   - Save
   - 等待2分钟

5. **访问游戏**
   ```
   https://你的用户名.github.io/math-game/
   ```
