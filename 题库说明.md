## 🎨 题目格式说明

每道题包含：

```json
{
  "id": 1,                           // 唯一ID
  "type": "addition",                // 题型
  "difficulty": 1,                   // 难度(1-4)
  "questionText": "1 + 1 = ?",       // 题目文字
  "num1": 1,                         // 第一个数
  "num2": 1,                         // 第二个数
  "correctAnswer": 2,                // 正确答案
  "wrongAnswers": [1, 3, 4]          // 错误选项
}
```

### 题型 (type)：
- `addition` - 加法
- `subtraction` - 减法
- `multiplication` - 乘法
- `division` - 除法
- `comparison` - 比大小

### 难度 (difficulty)：
- `1` - 简单（10以内）
- `2` - 中等（20以内）
- `3` - 较难（乘除法）
- `4` - 困难（高级题）
